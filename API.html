✧ API Project ✧


In my software engineering program (SEP), we tinkered and made a project based on APIs. My partners, Amelia, Nina, and I decided to group together to work on an idea of what we could do with our chosen API. While looking at the data set of popular baby names in NYC, we noticed that it only had females in it. This meant we could not cater to a wider audience and that we needed to specify where the questions were going to go. Along with gender, came data of the year of birth, ethnicity, child's first name, count, and rank. When starting to code, we were genuinely lost as none of us could figure out the for loop parameters. After some researching, we found the solution on how to loop through every object in the API to check if the user's input matched any of the data. With some time left, we decided to create a tiny extension that actually recommended the user a female baby name if they needed one. This was done by assigning each name to a certain number and having that user pick a number from one to a thousand. In the end, there were many times of frustration and "a ha moments" that really made us proud of this project. Going on, we would love to include more CSS in the javascript, if possible. We could add a slideshow of babyesque things or make the site "cuter" and user-friendly so it looks appealing to those who want a baby name.
